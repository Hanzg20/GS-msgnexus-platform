## 🎯 项目概述

MsgNexus 是一个专业的企业级消息管理平台，采用微服务架构，支持多租户、实时通信、智能监控等功能。

## 🛠️ 技术栈

### 前端技术
- **框架**: React 18 + TypeScript
- **状态管理**: Zustand
- **UI 组件**: 自定义组件库
- **动画**: Framer Motion
- **图表**: Recharts
- **样式**: CSS Modules + 自定义工具类

### 后端技术
- **运行时**: Node.js 18
- **框架**: Express.js
- **实时通信**: Socket.IO
- **数据库**: PostgreSQL + Redis
- **ORM**: Prisma
- **认证**: JWT

### 基础设施
- **容器化**: Docker
- **编排**: Kubernetes
- **监控**: Prometheus + Grafana
- **日志**: Winston
- **CI/CD**: GitHub Actions
